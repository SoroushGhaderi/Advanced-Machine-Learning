# Baselines Versus Ensemble Complexity

**Notebook:** `05_ensemble_learning.ipynb`

Random forests, voting, and stacking should be compared against a transparent baseline. An ensemble is valuable only when its incremental predictive or business value justifies extra latency, artifacts, dependencies, monitoring, and explanation burden.

The question is not whether an ensemble wins a leaderboard by a fraction, but whether it improves the deployed decision system.
