| [03 imbalanced learning](../notebooks/03_imbalanced_learning.ipynb) | Thresholds, costs, metrics, and realistic validation |
| [04 Optuna](../notebooks/04_optuna_hyperparameter_optimization.ipynb) | Development/validation/test discipline during tuning |
| [05 ensembles](../notebooks/05_ensemble_learning.ipynb) | Complexity trade-offs and policy-oriented evaluation |
| [06 anomaly extension](../notebooks/06_anomaly_detection_extension.ipynb) | Validation and monitoring for an optional extension |
| [07 production project](../notebooks/07_end_to_end_production_ml_project.ipynb) | Production contracts, serving consistency, and final evidence |
